프로젝트 개요 및 개발 상세 정의서

프로젝트명: Shape Link Puzzle

개발 목표:
이 게임은 특정 규칙을 만족하는 카드를 일렬 보드에 배치하면서 최대한 많은 카드를 배치하는 것을 목표로 한다. 사용자는 손패(Hand)에서 카드를 선택하여 보드에 배치할 수 있으며, 배치 조건을 충족해야만 놓을 수 있다.

게임 규칙:
1. 게임 보드는 가로로 1줄의 연결된 칸으로 구성된다.
2. 카드에는 세 가지 속성(배경색, 도형 형태, 도형 색상)이 있으며, 각각 3가지 종류가 존재한다.
3. 새로 놓을 카드는 이전 3개의 카드 중 2개의 카드와 비교하여 각 속성이 같거나 다를 때 배치할 수 있다.
(EX. 색상이 모두 일치, 도형은 모두 다름, 배경색은 모두 일치)
4. 카드를 배치하면 손패에 새로운 랜덤 카드가 추가된다.
5. 더 이상 배치할 수 없으면 게임 종료, 배치한 카드 개수가 점수로 반영된다.

기술 스택:
1. 프론트엔드: React + TypeScript
2. 상태관리: Zustand 또는 useState
3. UI: TailwindCSS      

주요 컴포넌트:
1. GameBoard.tsx: 보드 UI 및 카드 배치 검증
2. Hand.tsx: 플레이어 손패 관리   
3. Card.tsx: 개별 카드 UI 및 속성 표시
4. GameState.ts: 게임 상태 및 배치 검증 로직 관리
